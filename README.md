# Loan Management System

## Description
This is a Java-based Loan Management System developed as a final project for Programming 2 and Information Management subjects. The application allows users to:

- Register borrowers and manage user accounts.
- Record and track loan information.
- Calculate interest and total loan amounts based on loan size.
- Track loan payments, due dates, and penalties for late payments.

## Features
1. User Registration & Login – Secure registration and login system using text files.
2. Loan Information Form – Collects borrower details.
3. Loan Calculation– Calculates interest rates (10% for ≤50,000 PHP, 15% for >50,000 PHP) and total payable amounts.
4. Payment Management– Allows users to record payments and view remaining balances.
5. Penalty Tracking – Automatically calculates a 2% penalty for missed payments.
6. Dashboard – Users can view loan history and check for penalties.

## How to Run
1. Open the project in **IntelliJ IDEA** or any Java IDE.
2. Make sure all `.java` files and `loans.txt` / `users.txt` are in the same project folder.
3. Run `REGISTRATION_FORM.java` to start the application.
4. Navigate through the forms to register, input loan information, calculate payments, and view the dashboard.
